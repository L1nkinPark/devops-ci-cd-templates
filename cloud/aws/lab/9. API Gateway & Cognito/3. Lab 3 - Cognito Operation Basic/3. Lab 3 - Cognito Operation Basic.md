## II. Tổng quan bài Lab (Yêu cầu)

Bài Lab này hướng dẫn các thao tác cơ bản để thiết lập dịch vụ quản lý người dùng và xác thực **Amazon Cognito User Pool**:

1. **Khởi tạo Cognito User Pool:**
   * Tạo một User Pool có tên `test-user-pool-01` hỗ trợ đăng nhập bằng **Email** và **Username**.
   * Bật chức năng tự đăng ký (**Self-registration**) cho phép người dùng tự tạo tài khoản.
   * Yêu cầu các thuộc tính bắt buộc khi đăng ký là: **email** và **name**.
   * Thiết lập App Client dạng **Traditional web application** và đặt URL chuyển hướng (Return URL/Redirect URL) sau khi đăng nhập thành công là `https://h1eudayne.dev`.

2. **Quản trị người dùng (Tạo User bằng quyền Admin):**
   * Quản trị viên khởi tạo tài khoản thủ công trong User Pool với username là `h1eudayne`, email `voduchieu42@gmail.com` và đánh dấu email này là đã xác thực (**verified**).
   * Đặt mật khẩu tạm thời cho tài khoản.

3. **Kiểm tra đăng nhập bằng Hosted UI:**
   * Truy cập giao diện đăng nhập được Cognito cung cấp sẵn (**View login page**).
   * Thực hiện đăng nhập bằng user vừa tạo, đổi mật khẩu trong lần đăng nhập đầu tiên.
   * Kiểm tra chuyển hướng thành công về Return URL kèm theo Authorization Code / Identity Token.
