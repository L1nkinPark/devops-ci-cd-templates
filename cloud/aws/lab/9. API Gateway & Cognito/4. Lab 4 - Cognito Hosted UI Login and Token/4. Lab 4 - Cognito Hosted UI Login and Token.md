## II. Tổng quan bài Lab (Yêu cầu)

Bài Lab này hướng dẫn các thao tác thử nghiệm đăng ký, đăng nhập sử dụng giao diện dựng sẵn (**Hosted UI**) do Amazon Cognito cung cấp, và sử dụng script Python để đổi mã xác thực (Authorization Code) lấy JWT Tokens:

1. **Khởi tạo và cấu hình App Client:**
   * Tạo một App Client mới có tên `test-hosted-ui` dạng **Traditional web application**.
   * Thiết lập **Redirect URL** (Return URL) trỏ về trang web của bạn (ví dụ: `https://h1eudayne.dev`).
   * Bật tùy chọn tự động sinh **Client Secret**.

2. **Chỉnh sửa thông số App Client & Xử lý lỗi "Login pages unavailable":**
   * Cấu hình các Authentication Flows hợp lệ cho App Client.
   * Xử lý lỗi "Login pages unavailable" bằng cách đảm bảo đã tạo **Cognito Domain** và bật **Identity Provider** (Cognito user pool directory).

3. **Thực hiện Đăng nhập và lấy Authorization Code:**
   * Truy cập Hosted UI qua nút **View login page**.
   * Đăng nhập bằng tài khoản `h1eudayne` đã tạo ở Lab trước.
   * Lấy mã `code` xuất hiện trên thanh địa chỉ URL sau khi được redirect về trang web cá nhân.

4. **Đổi mã code lấy JWT Tokens:**
   * Cấu hình script Python `exchange-token.py` với các tham số: `client_id`, `client_secret`, `redirect_uri`, `token_endpoint`, và `authorization_code`.
   * Thực thi script `python ./exchange-token.py` để lấy các token xác thực: **ID Token**, **Access Token**, **Refresh Token**.
