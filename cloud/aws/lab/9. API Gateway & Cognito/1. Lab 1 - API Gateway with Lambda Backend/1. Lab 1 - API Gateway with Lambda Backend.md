## II. Tổng quan bài Lab (Yêu cầu)

Bài Lab này yêu cầu bạn xây dựng một Serverless API đơn giản có chức năng tính toán (Calculator) bằng cách kết hợp **Amazon API Gateway** và **AWS Lambda**:

1. **AWS Lambda (Backend):**
   * Khởi tạo một hàm Lambda chạy runtime **Python 3.12** với tên `test-calculator-function`.
   * Hàm Lambda nhận dữ liệu đầu vào chứa các tham số: `firstNum` (số thứ nhất), `secondNum` (số thứ hai), và `operator` (phép tính: `ADD`, `SUBSTRACT`, `MULTIPLE`, `DEVIDE`).
   * Thực hiện tính toán và trả về kết quả dưới dạng HTTP Response JSON.

2. **Amazon API Gateway (Entrypoint):**
   * Tạo một **REST API** trên API Gateway với tên `test-api`.
   * Tạo tài nguyên (Resource) và phương thức (Method) `POST` nhận request body từ client.
   * Định tuyến (forward) các yêu cầu từ Client tới Lambda backend xử lý bằng phương thức tích hợp **Non-Proxy Integration** (để giữ nguyên cấu trúc event truyền thẳng JSON body trực tiếp cho Lambda xử lý).
   * Triển khai (Deploy) API Gateway lên một Stage (ví dụ: `dev`) để sinh ra Endpoint URL công khai.

3. **Kiểm thử & Xác minh:**
   * Sử dụng công cụ kiểm thử API (Postman, cURL, v.v.) để gửi request dạng JSON tới API Gateway và kiểm tra kết quả trả về.
   * Xem và phân tích nhật ký thực thi (Execution logs) trong **Amazon CloudWatch Logs**.
