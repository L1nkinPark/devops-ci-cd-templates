# 10. Lab 2 - Install CloudWatch Agent

## I. Yêu cầu bài thực hành
Bài thực hành hướng dẫn cấu hình và cài đặt CloudWatch Agent trên máy chủ EC2 Linux nhằm thu thập các file logs hệ thống, logs ứng dụng web Apache và các chỉ số đo lường tùy chỉnh (Memory, Disk space).

Các công việc chính cần thực hiện:
1. Tạo một **IAM Role** cho phép dịch vụ EC2 kết nối và ghi dữ liệu log/metrics lên CloudWatch (`CloudWatchAgentServerPolicy`), sau đó gắn Role này vào instance EC2.
2. Tải và cài đặt gói phần mềm **Amazon CloudWatch Agent** trên máy chủ Linux.
3. Tạo tệp cấu hình `amazon-cloudwatch-agent.json` để định nghĩa:
   * Thu thập Log File của Apache Web Server: `/var/log/httpd/access_log`.
   * Thu thập Custom Metrics hệ thống: `mem_used_percent` (tỷ lệ RAM sử dụng) và `disk_used_percent` (tỷ lệ ổ đĩa sử dụng).
4. Khởi chạy CloudWatch Agent service trên EC2.
5. Truy cập CloudWatch Console để xác thực Logs đã được đẩy lên Log Group `/var/log/httpd/access_log` và các Custom Metrics hiển thị trong Custom Namespace `CWAgent`.
