# 19. Lab 5 - CloudTrail

## I. Yêu cầu bài thực hành
Bài thực hành hướng dẫn cấu hình tạo một Trail mới trong AWS CloudTrail để lưu trữ log cuộc gọi API ra S3 bucket và CloudWatch Logs.

Các công việc chính cần thực hiện:
1. Tạo một **Trail** mới trong AWS CloudTrail, cấu hình output log ra S3 bucket mới và CloudWatch Logs. (Lưu ý: Không bật Encrypt KMS khi làm lab).
2. Kiểm tra log group được tự động tạo trên CloudWatch.
3. Kiểm tra bucket và thư mục log được tạo trên S3.
4. Tạo một EC2 Instance để phát sinh event mới (`RunInstances`).
5. Kiểm tra log của event vừa tạo trên CloudWatch.
6. Tải và kiểm tra file log định dạng JSON trên S3.
7. Sử dụng Instance ID để tìm kiếm lại lịch sử sự kiện trên **CloudTrail Event history**.
