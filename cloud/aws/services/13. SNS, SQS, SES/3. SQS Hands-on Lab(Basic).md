# 3. SQS Hands-on Lab(Basic)

Bài thực hành này đã được di chuyển sang thư mục thực hành riêng (Labs) của repository để phân tách rõ ràng tài liệu lý thuyết (Services) và hướng dẫn thực hành chi tiết (Labs).

Để xem hướng dẫn thực hiện chi tiết từng bước bao gồm:
1. Truy cập trang quản trị SQS trên AWS Console.
2. Tạo standard queue với tên `test-queue-standard`.
3. Thử gửi 3-5 message với nội dung JSON/Text.
4. Thử nhận (poll) message.
5. Thao tác xóa message và Purge hàng đợi (xóa toàn bộ message).
