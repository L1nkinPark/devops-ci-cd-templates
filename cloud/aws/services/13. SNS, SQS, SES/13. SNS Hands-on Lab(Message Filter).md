# 13. SNS Hands-on Lab(Message Filter)

Bài thực hành này đã được di chuyển sang thư mục thực hành riêng (Labs) của repository để phân tách rõ ràng tài liệu lý thuyết (Services) và hướng dẫn thực hành chi tiết (Labs).

Để xem hướng dẫn thực hiện chi tiết từng bước bao gồm:
1. Tạo hoặc sử dụng lại Amazon SNS Topic loại Standard.
2. Tạo hàm AWS Lambda để làm subscriber.
3. Đăng ký Lambda Subscription và cấu hình Filter Policy `{"sendTo": ["LAMBDA"]}`.
4. Đăng ký SQS Subscription và cấu hình Filter Policy `{"sendTo": ["SQS"]}`.
5. Kiểm thử định tuyến tin nhắn bằng cách gửi tin nhắn kèm các Message Attributes khác nhau.
