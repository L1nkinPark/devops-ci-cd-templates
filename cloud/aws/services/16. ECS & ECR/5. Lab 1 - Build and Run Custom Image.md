# 5. Lab 1: Build và chạy custom image sử dụng Dockerfile

**Mục tiêu bài Lab:**
Trong bài lab này, bạn sẽ thực hành viết một Dockerfile để đóng gói một trang web HTML tĩnh chạy trên nền Web Server Apache (httpd). Bạn sẽ xây dựng (build) Docker Image từ Dockerfile đó và khởi chạy (run) một Docker Container cục bộ để kiểm tra hoạt động của ứng dụng.
