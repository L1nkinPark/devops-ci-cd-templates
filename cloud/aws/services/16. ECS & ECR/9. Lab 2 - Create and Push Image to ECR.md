# 9. Lab 2: Tạo và Push Image lên Amazon ECR

**Mục tiêu bài Lab:**
Trong bài lab này, bạn sẽ thực hành tạo một kho lưu trữ riêng tư (Private Repository) trên Amazon ECR, cấu hình khóa bảo mật và mã hóa AES-256. Sau đó, bạn sẽ thực hiện đăng nhập Docker client bằng access key thông qua lệnh CLI, gắn thẻ tag khớp với URI của repo, và đẩy (push) custom Docker image lên registry đám mây để lưu trữ.

## Tóm tắt các lệnh cốt lõi của bài Lab:

1. **Lệnh tạo repository:**
   ```bash
   aws ecr create-repository --repository-name my-httpd --region us-east-1
   ```

2. **Lệnh xác thực và đăng nhập Docker:**
   ```bash
   aws ecr get-login-password --region us-east-1 | docker login --username AWS --password-stdin <account-id>.dkr.ecr.us-east-1.amazonaws.com
   ```

3. **Lệnh gắn tag image:**
   ```bash
   docker tag my-httpd:latest <account-id>.dkr.ecr.us-east-1.amazonaws.com/my-httpd:latest
   ```

4. **Lệnh đẩy image lên AWS ECR:**
   ```bash
   docker push <account-id>.dkr.ecr.us-east-1.amazonaws.com/my-httpd:latest
   ```
