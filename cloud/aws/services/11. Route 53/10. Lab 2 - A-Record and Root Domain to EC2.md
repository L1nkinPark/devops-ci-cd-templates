# 10. Lab 2 - A-Record and Root Domain to EC2

## I. Yêu cầu bài thực hành
Bài thực hành hướng dẫn kết nối trực tiếp tên miền phụ (subdomain) và tên miền gốc (root domain) tới một máy chủ EC2 đang hoạt động bằng bản ghi loại A.

Các công việc chính cần thực hiện:
1. Khởi tạo một EC2 Instance chạy Linux và cài đặt dịch vụ máy chủ web Apache bằng User Data (Sử dụng script [lab3-user-data-meta-data.sh](file:///d:/Code/Deploy/cloud/aws/lab/1.%20EC2/3.%20Amazon%20EC2%20User%20Data%20and%20Metadata%20Lab/lab3-user-data-meta-data.sh)).
2. Tạo bản ghi loại **A-Record** trỏ subdomain (ví dụ: `testserver.h1eudayne.click`) về Public IP tĩnh của EC2.
3. Tạo bản ghi loại **A-Record** (hoặc bản ghi **ALIAS** trỏ về Load Balancer) cho tên miền gốc (Root Domain - ví dụ: `h1eudayne.click`).
4. Sử dụng công cụ dòng lệnh `nslookup` hoặc `dig` và truy cập trình duyệt để xác minh kết quả.
