# 2. Route 53 Features

Amazon Route 53 cung cấp 3 tính năng chính bao gồm:

## I. Register Domain name (Đăng ký tên miền)
Cho phép bạn tìm kiếm, mua và quản lý tên miền trực tiếp trên AWS Console, hỗ trợ tính năng tự động gia hạn (optional). Bạn cũng có thể đem một domain đã mua của bên thứ 3 (như GoDaddy, Namecheap,...) về và quản lý các cấu hình bản ghi trên Route 53 bằng cách thay đổi Name Servers của domain đó sang Route 53 Name Servers.

---

## II. DNS Routing (Định tuyến DNS)
Điều hướng lưu lượng truy cập Internet (internet traffic) tới một tài nguyên (resource) nhất định. Ví dụ:
* Địa chỉ IP của máy chủ ảo **Amazon EC2**.
* Bộ cân bằng tải **Application Load Balancer (ALB)**.
* Mạng phân phối nội dung **Amazon CloudFront**.
* Cổng API **Amazon API Gateway**.
* Cơ sở dữ liệu **Amazon RDS**.
* Các máy chủ và tài nguyên bên ngoài AWS.

---

## III. Health checking (Giám sát sức khỏe tài nguyên)
Tự động gửi các yêu cầu kiểm tra (requests) đến các resource định kỳ để kiểm tra tình trạng hoạt động. Tính năng này có thể kết hợp chặt chẽ với **AWS CloudWatch Alarm** để gửi cảnh báo tự động (qua SNS email/SMS) khi phát hiện resource bị lỗi hoặc không thể truy cập (unhealthy), hỗ trợ xây dựng hệ thống tự động khắc phục sự cố.
