# 7. Lab 2 – API Key & Usage Plan (API Gateway & Cognito Hands-on Lab)

Bài thực hành này đã được chuyển sang thư mục hướng dẫn triển khai (**lab**) của dự án để đảm bảo tính phân tách tài nguyên rõ ràng giữa phần tài liệu giới thiệu dịch vụ (Services) và hướng dẫn thực hành triển khai thực tế (Deploy).

Để xem toàn bộ hướng dẫn thực hành từng bước chi tiết (Hands-on Lab) bao gồm:
1. Khởi tạo một Usage Plan cấu hình Throttling (Rate/Burst) và Quota để quản lý tần suất gọi API.
2. Tạo một API Key tự động sinh ngẫu nhiên từ hệ thống.
3. Gán API Key và Stage `dev` của API `test-api` vào Usage Plan tương ứng.
4. Kích hoạt tính năng **API key required** trên method `POST /calculate` và tiến hành deploy lại API.
5. Thực hiện kiểm thử trên Postman để xác nhận request bị chặn 403 Forbidden nếu thiếu key và xử lý thành công 200 OK nếu đính kèm header `x-api-key`.
