# Hệ sinh thái API Gateway và Use Cases

## 1. Hệ sinh thái API Gateway

API Gateway là một service chủ yếu có nhiệm vụ nhận request của client sau đó forward (chuyển tiếp) tới các service phía sau. Dịch vụ này có thể tích hợp với rất nhiều thành phần khác trong hệ sinh thái AWS như Cognito (để xác thực), Lambda, DynamoDB, EC2, EKS, S3, Step Functions, IoT...

![API Gateway Ecosystem](../../../../images/aws/api_gateway_ecosystem.png)

## 2. Khi nào nên sử dụng API Gateway?

API Gateway phù hợp cho những bài toán và trường hợp sử dụng (use cases) sau:

- **Kiến trúc Microservices** sử dụng AWS Lambda làm backend (Serverless).
- **Backend API** cho hầu hết các use cases (như Web API, Mobile API, IoT).
- **Gateway nhận data trực tiếp từ client** sau đó lưu thẳng vào **DynamoDB** (Mô hình DB First, không cần qua compute layer).
- **Web Socket** cho những hệ thống yêu cầu giao tiếp theo thời gian thực (realtime communication) như ứng dụng chat, dashboard trực tuyến.
