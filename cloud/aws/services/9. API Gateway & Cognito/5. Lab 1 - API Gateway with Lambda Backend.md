# 5. Lab 1 – API Gateway sử dụng Lambda làm backend (API Gateway & Cognito Hands-on Lab)

Bài thực hành này đã được chuyển sang thư mục hướng dẫn triển khai (**lab**) của dự án để đảm bảo tính phân tách tài nguyên rõ ràng giữa phần tài liệu giới thiệu dịch vụ (Services) và hướng dẫn thực hành triển khai thực tế (Deploy).

Để xem toàn bộ hướng dẫn thực hành từng bước chi tiết (Hands-on Lab) bao gồm:
1. Khởi tạo một hàm Lambda chạy runtime Python 3.12 nhận các tham số đầu vào để thực hiện các phép tính cơ bản.
2. Xây dựng một REST API trên API Gateway với Resource và Method tương ứng.
3. Định tuyến và tích hợp Non-Proxy Integration để truyền trực tiếp dữ liệu từ API Gateway xuống Lambda.
4. Triển khai API lên Stage `dev` và thực hiện kiểm thử thành công bằng Postman.
