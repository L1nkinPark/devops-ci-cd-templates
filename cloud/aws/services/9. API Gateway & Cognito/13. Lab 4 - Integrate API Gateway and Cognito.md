# 13. Lab 4 – Kết hợp API Gateway & Cognito (API Gateway & Cognito Hands-on Lab)

Bài thực hành này đang trong quá trình biên soạn và chuẩn bị cập nhật tài liệu.

Nội dung dự kiến sẽ bao gồm:
1. Cấu hình Cognito User Pool làm Authorizer trên API Gateway REST API.
2. Thiết lập chuyển tiếp Token từ Client gửi đến API Gateway để kiểm tra hợp lệ.
3. Chuyển tiếp context thông tin user (như username, email) xuống Lambda backend.
