# Amazon API Gateway Pricing

API Gateway là một dịch vụ có chi phí nhàn rỗi (idle cost) bằng 0. Người dùng chỉ trả tiền cho khối lượng tài nguyên thực tế đã sử dụng. Cụ thể:

## 1. Đối với REST API

- **Số lượng request:** Chi phí được tính dựa trên số lượng yêu cầu gọi đến API (Ví dụ tại Singapore region: khoảng **$4.25 / 1 triệu requests**).
- **Data transfer out:** Chi phí truyền tải dữ liệu từ API Gateway ra bên ngoài internet (tính theo **$/GB**).
- **Caching:** Nếu có sử dụng bộ đệm API, chi phí sẽ tính theo dung lượng bộ đệm theo giờ (**GB/hour**).

## 2. Đối với WebSocket

- **Số lượng Message:** Tính phí dựa trên số lượng tin nhắn giao tiếp (Ví dụ: **$1.15 / 1 triệu messages** đối với các block dung lượng 32KB).
- **Connection minutes (Thời gian duy trì kết nối):** Tính dựa trên số phút kết nối được mở (Ví dụ: **$0.288 / 1 triệu connection minutes**).
